# p5 Websocket Demo

## Installation

To use this repository, clone it to your local computer. In your terminal, install the
server dependencies with:

```
npm install
```

Then, you can start the server with:

```
npm start
```

And navigate to [http://localhost:3000/](http://localhost:3000) and click and drag to draw on the
canvas. If you open multiple tabs, they'll all draw to the same canvas.
